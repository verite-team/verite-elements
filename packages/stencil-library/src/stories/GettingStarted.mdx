# Getting Started

Getting started with Verite Elements is as easy as including a script and dropping components into your HTML. No build steps. No dependencies.

## Install

```bash
npm install @verite/elements
```

Or use the CDN:

```html
<script src="https://unpkg.com/@verite/elements@latest/dist/verite-elements.js"></script>
```

## Basic Usage

```html
<verite-sign-in></verite-sign-in>
```

## Integrating with a Framework

Verite Elements works with any framework that supports custom elements. Here are some examples:

### React

```jsx
import { SignIn } from '@verite/elements'

<SignIn />
```

### Vue

```vue
<template>
  <SignIn />
</template>
```

### Angular

```tsx
import { SignIn } from '@verite/elements'

<SignIn />
```
